---
title: group
description: Generic grouping container.
---

A generic grouping element. Renders like `box` but provides semantic grouping meaning.

## Usage

```wire preview
(group :row :gap 8
  (button "Cancel" :ghost)
  (button "Save" :primary))
```

## Basic Examples

```wire preview
; Button group
(group :row :gap 8
  (button "Edit" :ghost)
  (button "Delete" :danger))

; Icon group
(group :row :gap 4
  (icon "bold")
  (icon "italic")
  (icon "underline"))
```


```wire preview

  (button "Day" :ghost :active)

  (button "Month" :ghost))
```
