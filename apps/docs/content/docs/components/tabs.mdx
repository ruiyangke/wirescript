Tab container for switching between views. Contains `tab` children.

## Usage

```wire preview
(tabs
  (tab "Overview" :active)
  (tab "Details")
  (tab "Settings"))
```

## Tab States

```wire preview
(tabs
  (tab "Active" :active)
  (tab "Normal"))
```

## With Navigation

```wire
(tabs
  (tab "Home" :to home)
  (tab "Profile" :to profile)
  (tab "Settings" :to settings))
```

## Tabs with Content

Tabs can contain children that render as panel content:

```wire preview:250
(tabs
  (tab "Overview" :active
    (text "Overview content here"))
  (tab "Details"
    (text "Details content here")))
```

## Examples

### Settings Page

```wire preview:550
(box :col :gap 24 :padding 24
  (text "Settings" :high)
  (tabs
    (tab "General" :active)
    (tab "Security")
    (tab "Notifications")
    (tab "Billing"))
  (card :col :gap 16 :padding 24
    (input "Display Name")
    (input "Email" :type email)
    (input "Bio" :type textarea :rows 3)

      (button "Save Changes" :primary))))
```
