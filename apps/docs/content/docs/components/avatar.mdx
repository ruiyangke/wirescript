---
title: avatar
description: User avatar with initials.
---

Displays a user avatar with initials derived from the content string.

## Usage

```wire preview
(avatar "John Doe")
```

## Sizes

```wire preview
(box :row :gap 12 :center
  (avatar "John Doe" :high)
  (avatar "John Doe" :medium)
  (avatar "John Doe" :low))
```

## Properties

<PropsTable element="avatar" />

## Examples

```wire preview
(box :row :gap 12
  (avatar "John Doe")
  (avatar "John")
  (avatar "AB"))
```
