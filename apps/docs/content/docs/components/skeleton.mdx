A placeholder element shown while content is loading.

## Usage

```wire preview
(skeleton)
```

## With Dimensions

```wire preview
(skeleton :width "200px" :height "20px")
(skeleton :width "100%" :height "16px")
(skeleton :width "48px" :height "48px")
```

## Properties

<PropsTable element="skeleton" />

Note: When no flags or dimensions are specified, a leaf skeleton defaults to text-line style (`:text`).

## Examples

### Circle and Text Variants

```wire preview:250
(box :col :gap 16
  ; Avatar placeholder
  (skeleton :circle :width "48px" :height "48px")

  ; Text line placeholder
  (skeleton :text)

  ; Custom dimensions
  (skeleton :width "200px" :height "20px"))
```

### Text Lines

```wire preview:250
(wire
  (screen main "Main" :desktop
    (box :col :gap 8 :padding 24
      (skeleton :height 24 :width "60%")
      (skeleton :height 16 :width "100%")
      (skeleton :height 16 :width "100%")
      (skeleton :height 16 :width "80%"))))
```

### Card Skeleton

```wire preview:200
(card :row :gap 16 :padding 16
  (skeleton :width 80 :height 80)
  (box :col :gap 8 :fill
    (skeleton :height 20 :width "70%")
    (skeleton :height 14)
    (skeleton :height 14 :width "50%")))
```

### User List Skeleton

```wire preview:400
(box :col :gap 0
  (repeat :count 5
    (box :row :gap 12 :padding 12
      (skeleton :width "40px" :height "40px")
      (box :col :gap 4 :fill
        (skeleton :height "16px" :width "60%")
        (skeleton :height "12px" :width "40%")))))
```

### Dashboard Skeleton

```wire preview:500
(wire
  (screen main "Main" :desktop
    (box :col :gap 24 :padding 24
      (box :row :gap 16
        (card :col :gap 8 :padding 20 :fill
          (skeleton :height 16 :width "50%")
          (skeleton :height 32 :width "70%"))
        (card :col :gap 8 :padding 20 :fill
          (skeleton :height 16 :width "50%")
          (skeleton :height 32 :width "70%"))
        (card :col :gap 8 :padding 20 :fill
          (skeleton :height 16 :width "50%")
          (skeleton :height 32 :width "70%")))
      (card :col :gap 16 :padding 20
        (skeleton :height 20 :width "30%")
        (skeleton :height 250 :width "100%")))))
```
