A placeholder element shown while content is loading.

## Usage

```wire preview
(skeleton)
```

## With Dimensions

```wire preview
(skeleton :width "200px" :height "20px")
(skeleton :width "100%" :height "16px")
(skeleton :width "48px" :height "48px")
```

## Properties

<PropsTable element="skeleton" />

Note: When no flags or dimensions are specified, a leaf skeleton defaults to text-line style (`:text`).

## Examples

### Circle and Text Variants

```wire preview:250
(box :col :gap 16
  ; Avatar placeholder
  (skeleton :circle :width "48px" :height "48px")

  ; Text line placeholder
  (skeleton :text)

  ; Custom dimensions
  (skeleton :width "200px" :height "20px"))
```

### Text Lines

```wire preview
(box :col :gap 8
  (skeleton :height "24px" :width "60%")
  (skeleton :height "16px")
  (skeleton :height "16px")
  (skeleton :height "16px" :width "80%"))
```

### Card Skeleton

```wire preview:300
(card :col :gap 12 :padding 16
  (skeleton :height "150px")
  (skeleton :height "20px" :width "70%")
  (skeleton :height "16px")
  (skeleton :height "36px" :width "100%"))
```

### User List Skeleton

```wire preview:400
(box :col :gap 0
  (repeat :count 5
    (box :row :gap 12 :padding 12
      (skeleton :width "40px" :height "40px")
      (box :col :gap 4 :fill
        (skeleton :height "16px" :width "60%")
        (skeleton :height "12px" :width "40%")))))
```

### Dashboard Skeleton

```wire preview:400
(box :col :gap 24
  (box :row :gap 16
    (card :padding 20 :fill
      (skeleton :height "16px" :width "50%")
      (skeleton :height "32px" :width "70%"))
    (card :padding 20 :fill
      (skeleton :height "16px" :width "50%")
      (skeleton :height "32px" :width "70%"))
    (card :padding 20 :fill
      (skeleton :height "16px" :width "50%")
      (skeleton :height "32px" :width "70%")))
  (card :col :gap 16 :padding 20
    (skeleton :height "20px" :width "30%")
    (skeleton :height "300px")))
```

### Article Skeleton

```wire preview:300

  (skeleton :height "40px" :width "80%")
  (box :row :gap 8
    (skeleton :width "100px" :height "16px")

  (skeleton :height "200px")
  (skeleton :height "16px")
  (skeleton :height "16px")

```

### Profile Skeleton

```wire preview:300
(box :col :center :gap 16 :padding 24
  (skeleton :width "80px" :height "80px")
  (skeleton :height "24px" :width "150px")
  (skeleton :height "16px" :width "100px")
  (box :row :gap 8
    (skeleton :width "80px" :height "36px")
    (skeleton :width "80px" :height "36px")))
```

### Container Skeleton

Skeleton can act as a container with layout props when it has children:

```wire preview:200
(skeleton :row :gap 12 :padding 16
  (skeleton :circle :width "48px" :height "48px")
  (skeleton :col :gap 4 :fill
    (skeleton :height "16px" :width "60%")
    (skeleton :height "12px" :width "40%")))
```
