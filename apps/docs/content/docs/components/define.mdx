---
title: define
description: Create reusable components with parameters.
---

Define reusable components with parameters that can be used throughout your wireframes.

## Usage

```wire
(define component-name (param1 param2)
  (box :row
    (text $param1)
    (text $param2)))
```

## Calling Components

Always use named arguments:

```wire
(component-name :param1 "Hello" :param2 "World")
```

## Rules

1. Parameters are referenced with `$param` in the body
2. Calls must use named arguments (`:param value`)
3. Component names cannot shadow built-in elements

## Examples

### Stat Card

```wire preview:300
(wire
  (define stat-card (label value)
    (card :col :padding 16 :gap 8
      (text $label :low)
      (text $value :high)))

  (screen dashboard "Dashboard" :desktop
    (box :row :gap 16 :padding 24
      (stat-card :label "Users" :value "1,234")
      (stat-card :label "Revenue" :value "$5,678")
      (stat-card :label "Orders" :value "890"))))
```

### User Card

```wire preview:350
(wire
  (define user-card (name email role)
    (card :row :gap 16 :padding 16
      (avatar $name)
      (box :col :gap 4 :fill
        (text $name :high)
        (text $email :low))
      (badge $role)))

  (screen users "Users" :desktop
    (box :col :gap 8 :padding 24
      (user-card :name "Alice" :email "alice@example.com" :role "Admin")
      (user-card :name "Bob" :email "bob@example.com" :role "Editor"))))
```

### Nav Item

```wire preview:400
(wire
  (define nav-item (label icon target)
    (button $label :ghost :full :start :icon $icon :to $target))

  (layout app-layout
    (box :row :full
      (nav :col :width "200px" :padding 16 :gap 4
        (nav-item :label "Dashboard" :icon "house" :target dashboard)
        (nav-item :label "Projects" :icon "folder" :target projects)
        (nav-item :label "Settings" :icon "settings" :target settings))
      (slot)))

  (screen dashboard "Dashboard" :desktop :layout app-layout
    (box :col :padding 24
      (text "Dashboard Content" :high))))
```
