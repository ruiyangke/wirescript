title: header

---

A semantic header element with a bottom border, typically used for page headers and navigation bars.

## Usage

```wire preview
(header :row :between :padding 16
  (text "Logo" :high)
  (nav :row :gap 16
    (text "Home")
    (text "About")))
```

## Basic Examples

```wire preview
; Simple header
(header :padding 16
  (text "My App" :high))

; Header with navigation
(header :row :between :padding 16
  (text "Logo" :high)
  (box :row :gap 16
    (text "Home" :to home)
    (text "About" :to about)
    (text "Contact" :to contact)))
```

## Examples

### App Header

```wire preview
(header :row :between :padding 16
  (box :row :gap 12 :center
    (icon "box")
    (text "AppName" :high))
  (box :row :gap 16
    (text "Dashboard" :to dashboard)
    (text "Settings" :to settings)
    (avatar "JD")))
```

### Header with Search

```wire preview
(header :row :between :padding 16
  (text "Logo" :high)
  (input "Search" :type search :width "300px")
  (box :row :gap 12
    (icon "bell")
    (avatar "User")))
```

### Sticky Header

```wire preview
(header :row :between :padding 16 :sticky :top 0
  (text "Sticky Header" :high)
  (button "Action" :primary))
```

### Mobile Header

```wire preview
(header :row :between :padding 12
  (icon "menu" :to "#mobile-nav")
  (text "App" :high)
  (icon "bell"))
```
