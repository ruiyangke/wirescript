A small overlay that appears near a trigger element. Defined inside screens and triggered by `:to "#id"`.

## Usage

```wire preview:400
(wire
  (screen main "Main" :desktop
    (box :col :center :full :padding 24
      (box :row :center :gap 8
        (text "Hover for more info")
        (icon "info" :to "#info")))

    (popover :id "info"
      (box :col :gap 8 :padding 12
        (text "Additional Information" :high)
        (text "This popover provides extra context." :low)))))
```

## Opening a Popover

Use a `button` or clickable `text` with `:to "#id"`:

```wire
(button "Info" :ghost :to "#info")
(text "Learn more" :to "#info")
```

## Flags

| Flag | Effect |
|------|--------|
| `:open` | Popover is initially open |
| `:row` | Horizontal layout |
| `:col` | Vertical layout |
| `:center` | Center alignment |
| `:start` | Start alignment |
| `:end` | End alignment |
| `:between` | Space between |
| `:full` | Full size |
| `:fill` | Fill available space |

## Properties

| Property | Type | Example |
|----------|------|---------|
| `:id` | string | `:id "info"` |
| `:gap` | number | `:gap 8` |
| `:padding` | number | `:padding 12` |
| `:width` | number | `:width 250` |
| `:height` | number | `:height 200` |

## Examples

### Initially Open Popover

```wire preview:400
(wire
  (screen main "Main" :desktop
    (box :col :center :full :padding 24
      (box :row :center :gap 8
        (text "Hover for more info")
        (icon "info" :to "#info")))

    (popover :id "info" :open
      (box :col :gap 8 :padding 12
        (text "Information" :high)
        (text "This popover opens automatically." :low)))))
```

### User Preview

```wire preview:450
(wire
  (screen main "Main" :desktop

    (popover :id "user-preview"

```

### Color Picker Popover

```wire preview:400
(wire
  (screen main "Main" :desktop
    (box :col :center :full :padding 24
      (box :row :center :gap 8
        (text "Color:")
        (button "Select" :secondary :to "#color-picker")))

    (popover :id "color-picker"
      (box :col :gap 12 :padding 12
        (text "Choose Color" :low)
        (box :grid :cols 5 :gap 8
          (badge "")
          (badge "" :primary)
          (badge "" :success)
          (badge "" :warning)
          (badge "" :danger))))))
```
