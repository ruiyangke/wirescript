A small overlay that appears near a trigger element. Defined inside screens and triggered by `:to "#id"`.

## Usage

```wire preview:400
(screen main "Main" :desktop
  (box :col :padding 24
    (box :row :gap 8
      (text "Hover for more info")
      (icon "info" :to "#info")))

  (popover :id "info"
    (box :col :gap 8 :padding 12
      (text "Additional Information" :high)
      (text "This popover provides extra context." :low))))
```

## Opening a Popover

Use a `button` or clickable `text` with `:to "#id"`:

```wire
(button "Info" :ghost :to "#info")
(text "Learn more" :to "#info")
```

## Flags

| Flag | Effect |
|------|--------|
| `:open` | Popover is initially open |
| `:top` | Position above trigger |
| `:bottom` | Position below trigger |
| `:left` | Position to left of trigger |
| `:right` | Position to right of trigger |
| `:row` | Horizontal layout |
| `:col` | Vertical layout |
| `:grid` | Grid layout |
| `:center` | Center alignment |
| `:start` | Start alignment |
| `:end` | End alignment |
| `:between` | Space between |
| `:full` | Full size |
| `:fill` | Fill available space |

## Properties

| Property | Type | Example |
|----------|------|---------|
| `:id` | string | `:id "info"` |
| `:gap` | number | `:gap 8` |
| `:padding` | number | `:padding 12` |
| `:width` | any | `:width "250px"` |
| `:height` | any | `:height "200px"` |

## Examples

### Initially Open Popover

```wire preview:400
(screen main "Main" :desktop
  (box :col :padding 24
    (box :row :gap 8
      (text "Hover for more info")
      (icon "info" :to "#info")))

  (popover :id "info" :open
    (box :col :gap 8 :padding 12
      (text "Information" :high)
      (text "This popover opens automatically." :low))))
```

### Info Popover

```wire preview:400
(screen main "Main" :desktop
  (box :row :gap 8
    (text "Status")
    (icon "info" :to "#status-info"))

  (popover :id "status-info"
    (box :col :gap 8 :padding 12
      (text "Status Information" :high)
      (text "Your account is currently active and in good standing." :low))))
```

### User Preview

```wire preview:450
(screen main "Main" :desktop
  (box :row :gap 8
    (avatar "JD" :to "#user-preview")
    (text "John Doe"))

  (popover :id "user-preview"
    (box :col :gap 12 :padding 16
      (box :row :gap 12
        (avatar "John Doe")
        (box :col :gap 4
          (text "John Doe" :high)
          (text "Product Designer" :low)))
      (text "Building great products at Acme Inc." :low)
      (box :row :gap 8
        (button "Follow" :primary)
        (button "Message" :secondary)))))
```

### Color Picker Popover

```wire preview:400
(screen main "Main" :desktop
  (box :row :gap 8
    (text "Color:")
    (button "Select" :secondary :to "#color-picker"))

  (popover :id "color-picker"
    (box :col :gap 12 :padding 12
      (text "Choose Color" :low)
      (box :grid :cols 5 :gap 8
        (badge "")
        (badge "" :primary)
        (badge "" :success)
        (badge "" :warning)
        (badge "" :danger)))))
```

### Quick Actions Popover

```wire preview:400
(screen main "Main" :desktop

      (button "New Note" :ghost :full :start)
      (button "New Event" :ghost :full :start))))
```
