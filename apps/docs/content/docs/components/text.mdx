title: text

---

Display text with various emphasis levels and optional navigation.

## Usage

```wire preview
(text "Hello World")
```

## Emphasis Levels

```wire preview:250
(box :col :gap 8
  (text "Heading" :high)
  (text "Body text")
  (text "Caption" :low))
```

## Variants

```wire preview:250
(box :col :gap 8
  (text "Error message" :danger)
  (text "Success!" :success)
  (text "Warning" :warning)
  (text "Info" :info))
```

## Navigation

Text can be clickable:

```wire
(text "Go to settings" :to settings)
(text "Open modal" :to "#my-modal")
```

### Linked Text

```wire preview:250
(box :col :gap 4
  (text "Learn more about our features" :to features)
```
