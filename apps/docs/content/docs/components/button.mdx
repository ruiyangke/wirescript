Interactive button with multiple visual variants and states.

## Usage

```wire preview
(button "Click me")
```

## Variants

```wire preview
(box :row :gap 8
  (button "Primary" :primary)
  (button "Secondary" :secondary)
  (button "Ghost" :ghost)
  (button "Danger" :danger))
```

## States

```wire preview
(box :row :gap 8
  (button "Disabled" :primary :disabled)
  (button "Loading" :primary :loading)
  (button "Active" :ghost :active))
```

## Sizing

```wire preview
(button "Full Width" :primary :full)
```

## Navigation

```wire
; Navigate to screen
(button "Go to Dashboard" :primary :to dashboard)

; Open overlay
(button "Open Modal" :ghost :to "#my-modal")

; Close overlay
(button "Close" :ghost :to :close)
```
