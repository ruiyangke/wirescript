title: section
description: Semantic section container.
---

A semantic section element. Renders like `box` but provides semantic meaning. Content string is the section name.

## Usage

```wire preview
(section "hero"
  (text "Welcome"))
```

## Basic Examples

```wire preview:400
; Named section
(section "hero" :col :center :padding 48
  (text "Welcome" :high)
  (text "Build something great"))

; Features section
(section "features" :col :gap 24 :padding 32
  (text "Features" :high)
  (box :grid :cols 3 :gap 16
    (card :padding 16 (text "Feature 1"))
    (card :padding 16 (text "Feature 2"))
    (card :padding 16 (text "Feature 3"))))
```
