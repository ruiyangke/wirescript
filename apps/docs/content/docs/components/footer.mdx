A semantic footer element with a top border, typically used for page footers.

## Usage

```wire preview
(footer :center :padding 24
  (text "© 2024 Company" :low))
```

## Basic Examples

```wire preview:250
; Simple footer
(footer :center :padding 16
  (text "© Company Inc." :low))

; Footer with links
(footer :row :between :padding 24
  (text "© 2024 Company")
  (box :row :gap 16
    (text "Privacy")
    (text "Terms")
    (text "Contact")))
```
