---
title: divider
description: Horizontal or vertical divider line.
---

A line to visually separate content, either horizontally (default) or vertically.

## Usage

```wire preview
(divider)
```

## Vertical Divider

Use `:vertical` in row layouts:

```wire preview:200
(box :row :gap 8 :padding 16
  (text "Left")
  (divider :vertical)
  (text "Right"))
```

## Examples

### Separating Sections

```wire preview:250
(box :col :gap 16
  (text "Section 1" :high)
  (text "Content for section 1")
  (divider)
  (text "Section 2" :high)
  (text "Content for section 2"))
```

### In Navigation

```wire preview:250
(nav :col :gap 4 :padding 16
  (button "Dashboard" :ghost :full :start)
  (button "Projects" :ghost :full :start)
  (divider)
  (button "Settings" :ghost :full :start)
  (button "Logout" :ghost :full :start :danger))
```

### In Lists

```wire preview:250
(list :col :gap 0
  (box :padding 12 (text "Item 1"))
  (divider)
  (box :padding 12 (text "Item 2"))
  (divider)
  (box :padding 12 (text "Item 3")))
```

### In Cards

```wire preview:300
(card :col :padding 0
  (box :padding 16
    (text "Card Title" :high))
  (divider)
  (box :padding 16
    (text "Card content goes here"))
  (divider)
  (box :row :end :padding 12 :gap 8
    (button "Cancel" :ghost)
    (button "Save" :primary)))
```

### In Toolbars

```wire preview:200
(wire
  (screen main "Main" :desktop
    (box :col :center :full :padding 24
      (card :row :center :gap 12 :padding 8
        (group :row :gap 2
          (button :ghost :icon "bold")
          (button :ghost :icon "italic"))
        (divider :vertical)
        (group :row :gap 2
          (button :ghost :icon "align-left")
          (button :ghost :icon "align-right"))
        (divider :vertical)
        (group :row :gap 2
          (button :ghost :icon "list"))))))
```

### In Dropdowns

```wire preview:360
(dropdown :id "menu"
  (button "Edit" :ghost :full)
  (button "Duplicate" :ghost :full)
  (divider)
  (button "Archive" :ghost :full)
  (divider)
  (button "Delete" :danger :ghost :full))
```

### In Forms

```wire preview:600
(form :col :gap 16
  (text "Account Info" :high)
  (input "Email" :type email)
  (input "Password" :type password)
  (divider)
  (text "Profile" :high)
  (input "Display Name")
  (input "Bio" :type textarea :rows 3)
  (divider)
  (button "Save Changes" :primary))
```
