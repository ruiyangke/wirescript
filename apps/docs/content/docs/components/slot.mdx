---
title: slot
description: Layout placeholder for screen content.
---

A placeholder element used in layouts to mark where screen content should be inserted.

## Usage

```wire
(layout my-layout
  (header (text "Header"))
  (slot)
  (footer (text "Footer")))
```

## How It Works

When a screen uses a layout, the screen's content replaces the `(slot)` element:

```wire preview:300
(wire
  (layout app-layout
    (box :row :full
      (nav :col :width 200 :padding 16
        (text "Navigation" :high))
      (box :fill
        (slot))))

  (screen home "Home" :desktop :layout app-layout
    (card :col :padding 24 :gap 8
      (text "Page Content" :high)
      (text "This content replaces the slot" :low))))
```

## Examples

### Basic Layout

```wire preview:400
(wire
  (layout page-layout
    (box :col :full
      (header :row :between :center :padding 16
        (text "Logo" :high)
        (nav :row :gap 16
          (text "Home")
          (text "About")))
      (slot)
      (footer :row :center :padding 16

  (screen main "Main" :desktop :layout page-layout
    (box :col :center :fill :gap 8
      (text "Page Content" :high)
      (text "This content replaces the slot" :low))))
```

### Sidebar Layout

```wire preview:450
(wire
  (layout sidebar-layout
    (box :row :full
      (nav :col :width 200 :padding 16 :gap 8
        (text "App Name" :high)
        (divider)
        (button "Dashboard" :ghost :full :start)
        (button "Projects" :ghost :full :start)
        (button "Settings" :ghost :full :start))
      (box :col :fill
        (slot))))

  (screen dashboard "Dashboard" :desktop :layout sidebar-layout
    (box :col :padding 24 :gap 16
      (text "Dashboard" :high)
      (text "Welcome back! Here's your overview." :low))))
```

### Dashboard Layout

```wire preview:500
(wire
  (layout dashboard-layout
    (box :col :full
      (header :row :between :center :padding 16
        (text "Dashboard" :high)
        (box :row :center :gap 12
          (icon "bell")
          (avatar "User")))
      (box :row :fill
        (nav :col :width 180 :padding 16 :gap 4
          (button "Overview" :ghost :full :start)
          (button "Analytics" :ghost :full :start))
        (slot))))

  (screen overview "Overview" :desktop :layout dashboard-layout
    (box :col :padding 24 :gap 16
      (text "Overview" :high)
      (box :row :gap 16
        (card :col :gap 8 :padding 16 :fill
          (text "Users" :low)
          (text "1,234" :high))
        (card :col :gap 8 :padding 16 :fill
          (text "Revenue" :low)
          (text "$5,678" :high))))))
```
