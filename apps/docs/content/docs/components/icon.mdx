Displays an icon from the [Lucide](https://lucide.dev) icon set. The content string is the icon name.

## Usage

```wire preview
(icon "home")
```

## Examples

### Navigation Bar

```wire preview
(header :row :between :padding 16
  (icon "menu")
  (text "App" :high)
  (box :row :gap 12
    (icon "search")
    (icon "bell")
    (icon "user")))
```

### Bottom Navigation

