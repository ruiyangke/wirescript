Displays an icon from the [Lucide](https://lucide.dev) icon set. The content string is the icon name.

## Usage

```wire preview
(icon "house")
```

## Common Icons

## Examples

### Navigation Bar

```wire preview:200
(wire
  (screen main "Main" :desktop
    (box :col :full
      (header :row :between :center :padding 16
        (icon "menu" :to "#nav")
        (text "App" :high)
        (box :row :center :gap 12
          (icon "search")
          (icon "bell")
          (icon "user")))
      (box :fill :padding 24
        (text "Page content" :low)))))
```

### Bottom Navigation

```wire preview:450
(wire
  (screen main "Main" :mobile
    (box :col :full
      (box :fill :padding 16
        (box :col :gap 12
          (text "Home" :high)
          (text "Welcome to the app" :low)
          (card :padding 16 (text "Featured content"))))
      (nav :row :around :padding 12
        (box :col :center :gap 4
          (icon "house")
          (text "Home" :low))
        (box :col :center :gap 4
          (icon "search")
          (text "Search" :low))
        (box :col :center :gap 4
          (icon "heart")
          (text "Saved" :low))
        (box :col :center :gap 4
          (icon "user")
          (text "Profile" :low))))))
```
