---
title: metric
description: Key metric with optional trend indicator.
---

Displays a key metric with label, value, and optional change/trend indicators.

## Usage

```wire preview
(metric :label "Users" :value "1,234")
```

## With Trend

```wire preview:250
(metric
  :label "Revenue"
  :value "$45,678"
  :change "+12.5%"
  :trend up)

(metric
  :label "Issues"
  :value "89"
  :change "-5%"
  :trend down)
```

## Properties

<PropsTable element="metric" />

Note: The content string can be used as an alternative to `:label`. For example, `(metric "Revenue" :value "$1,234")` is equivalent to `(metric :label "Revenue" :value "$1,234")`.

Note: Variant flags apply color to the value text. For example, `:success` makes the value green, `:danger` makes it red.

## Examples

### With Variant Colors

```wire preview
(box :row :gap 24
  (metric :label "Growth" :value "+24%" :success)
  (metric :label "Errors" :value "12" :danger)
  (metric :label "Warning" :value "3" :warning))
```

### Dashboard Metrics

```wire preview:250
(box :row :gap 16
  (metric :label "Total Users" :value "12,345" :change "+12%" :trend up)
  (metric :label "Revenue" :value "$45,678" :change "+8%" :trend up)
  (metric :label "Orders" :value "890" :change "-3%" :trend down)
  (metric :label "Conversion" :value "2.4%" :change "+0.5%" :trend up))
```

### Metric Cards

```wire preview:250

```
