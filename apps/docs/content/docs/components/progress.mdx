A progress bar showing completion percentage.

## Usage

```wire preview:150
(progress :value 75)
```

## Basic Examples

```wire preview:250
(box :col :gap 16 :padding 24
  (box :col :gap 4
    (text "With value" :low)
    (progress :value 75))
  (box :col :gap 4
    (text "With max" :low)
    (progress :value 30 :max 100))
  (box :col :gap 4
    (text "Indeterminate" :low)
    (progress)))
```

### Upload Progress

```wire preview:200
(box :col :padding 24
  (card :col :gap 12 :padding 16
    (box :row :between :center
      (text "Uploading file.zip")

```

### Multi-step Progress

```wire preview:200
(box :col :gap 16 :padding 24
  (progress :value 33)
  (box :row :between

```
