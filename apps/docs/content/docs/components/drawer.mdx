---
title: drawer
description: Slide-in panel.
---

A slide-in panel from the left or right side. Defined inside screens and triggered by `:to "#id"`.

## Usage

```wire preview:400
(wire
  (screen main "Main" :desktop
    (box :col :center :full
      (button "Open Drawer" :primary :to "#my-drawer"))

    (drawer :id "my-drawer" :right
      (box :col :gap 16 :padding 24
        (box :row :between
          (text "Drawer Content" :high)
          (icon "x" :to :close))
        (text "This is a slide-in panel.")))))
```

## Position

```wire
(screen example "Example" :desktop
  (box :padding 16
    (text "Click buttons to open drawers"))

  ; Right side (default)
  (drawer :id "settings" :right
    (text "Settings"))

  ; Left side
  (drawer :id "menu" :left
    (text "Menu")))
```

## Flags

| Flag | Effect |
|------|--------|
| `:open` | Drawer is initially open |
| `:left` | Drawer slides from left |
| `:right` | Drawer slides from right |
| `:top` | Drawer slides from top |
| `:bottom` | Drawer slides from bottom |
| `:row` | Horizontal layout |
| `:col` | Vertical layout |
| `:grid` | Grid layout |
| `:center` | Center alignment |
| `:start` | Start alignment |
| `:end` | End alignment |
| `:between` | Space between |
| `:full` | Full size |
| `:fill` | Fill available space |

## Properties

| Property | Type | Example |
|----------|------|---------|
| `:id` | string | `:id "settings"` |
| `:title` | string | `:title "Settings"` |
| `:position` | string | `:position "left"` |
| `:gap` | number | `:gap 16` |
| `:padding` | number | `:padding 24` |
| `:width` | number | `:width 350` |
| `:height` | any | `:height "100%"` |

Note: Default width is 400px if not specified.

## Examples

### Initially Open Drawer

```wire preview:400
(wire
  (screen main "Main" :desktop
    (box :col :full
      (header :row :between :padding 16
        (text "Dashboard" :high)
        (button :ghost :icon "menu" :to "#sidebar"))
      (box :col :padding 24 :gap 16
        (text "Welcome back!" :high)
        (text "Your sidebar is open by default." :low)))
    (drawer :id "sidebar" :left :open :width 220
      (box :col :gap 8 :padding 16
        (box :row :between
          (text "Menu" :high)
          (icon "x" :to :close))
        (divider)
        (button "Dashboard" :ghost :full :start :icon "house")
        (button "Projects" :ghost :full :start :icon "folder")
        (button "Settings" :ghost :full :start :icon "settings")))))
```

### Settings Drawer

### Mobile Navigation

```wire preview:500
(wire
```

### Filter Drawer

```wire preview:500
(wire
  (screen products "Products" :desktop
    (box :row :between :padding 16
      (text "Products" :high)
      (button "Filters" :secondary :to "#filters"))

    (drawer :id "filters" :right :width 320
      (box :col :gap 16 :padding 24
        (box :row :between

          (icon "x" :to :close))
        (divider)
        (input "Category" :type select :options "All,Electronics,Clothing")
        (input "Price Range" :type slider :min 0 :max 1000)

        (input "On Sale" :type checkbox)
        (box :row :gap 12
          (button "Clear" :ghost :fill)
          (button "Apply" :primary :fill))))))
```

### Cart Drawer

```wire preview:500
(wire
  (screen shop "Shop" :desktop
    (header :row :between :padding 16
      (text "Shop" :high)
      (box :row :gap 4

        (badge "3" :primary)))

    (drawer :id "cart" :right :width 400
      (box :col :full
        (box :row :between :padding 16
          (text "Your Cart" :high)

        (divider)
        (scroll :col :gap 0 :fill
          (box :row :gap 12 :padding 16
            (image "Product" :width 80 :height 80)
            (box :col :gap 4 :fill
              (text "Product Name")
              (text "$29.00" :low))
            (icon "trash"))
          (divider)
          (box :row :gap 12 :padding 16
            (image "Product" :width 80 :height 80)
            (box :col :gap 4 :fill
              (text "Another Product")
              (text "$49.00" :low))
            (icon "trash")))

        (box :col :gap 12 :padding 16
          (box :row :between
            (text "Total")

```
